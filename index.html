About Golden Launcher

Golden Launcher is a fork of Copper Launcher built from a phone. About Me, ive alwyas wanted to contribute to the PojavLauncher Team and it was one of my dreams to make this kinds of launchers. I first known about the PojavLauncher back in 2021, i always used it i didnt want to play the bedrock Edition. It was a simple YouTube typo that i accidentally type "Minecraft Java" on the youtube search, then i scrolled untill i found a tutorial for Pojav Launcher (Im not Kidding, this is a real story). When the PojavLauncher Team Archived the repository, i couldnt do what i wanted. 2026 of April, i started working on Golden Launcher but it was not buult from source and more like a apk Mod than a real launcher. Then, i learned about how to actually use github so, i found out about the GitHub plugin for Acode, a VSCode like IDE for Android.

  • Mod support (Fabric & Forge)
  • Built from source and made for Low End Devices!
  • Supports Multiple renderers like, Krypton, Mobile Glues, and LTW
  • 🇵🇭 Made in the Philippines
